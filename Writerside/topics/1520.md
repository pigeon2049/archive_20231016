# 硕贝德股价六个交易日翻倍：公司和华为主要是天线业务有合作 公司正在研发6G卫通天线

**2023-10-16 05:09**

【硕贝德股价六个交易日翻倍：公司和华为主要是天线业务有合作 公司正在研发6G卫通天线】财联社10月16日电，在华为和6G概念的加持下，硕贝德上周五20CM涨停，6个交易日股价翻倍。今天早盘，股价继续冲高涨超8%。记者以投资者身份致电硕贝德投资者热线，公司人员表示，不清楚股价上涨原因。公司和华为主要是天线业务有合作。另外，在6G方面，公司正在研发卫通天线，但卫通天线没有批量出货。 (第一财经)
